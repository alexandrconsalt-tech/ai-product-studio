# Framework

Framework описывает признанный Product, AI Engineering или Architecture framework.

Сущность хранится отдельно и используется по ссылке из Product, Architecture, Review и KnowledgeModule.

Validation rules:

- `id` обязателен;
- `name` не пустой;
- `category` выбирается из фиксированного списка;
- `source` указывает источник framework;
- `version` фиксирует версию описания.
