🔑 API-ключи▶
Где взять ключи и куда они уходят
Anthropic-ключ создаётся в console.anthropic.com/settings/keys, OpenAI-ключ — в platform.openai.com/api-keys.
Каждый ключ используется только для шагов со своим вендором (см. поле «Модель» у шага) и хранится ТОЛЬКО в localStorage этого браузера (pipelineLabV3.anthropicApiKey, pipelineLabV3.openaiApiKey) — отправляется напрямую в api.anthropic.com / api.openai.com соответственно, никуда больше. Никогда не публикуйте эту страницу с сохранёнными ключами посторонним и не заходите с чужого компьютера, не удалив ключи после теста.

П Пайплайн▶
Переменные в промтах
{{transcript}} — текст транскрипции
{{ctx.КЛЮЧ}} — результат шага с этим ключом (JSON)
{{ctx.facts.budget}} — путь внутри результата
{{ctx.stored_files}} — файлы из «Хранилище входящих данных» (имя, формат; текст доступен только для .txt/.svg)
Сервисные и код-шаги (stt, validate, store, gate, crm) выполняются детерминированно, без модели.
