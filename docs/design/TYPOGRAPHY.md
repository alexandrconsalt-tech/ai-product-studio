# Typography

## Scale

| Token | Назначение |
|---|---|
| Heading XL | Главные заголовки workspace |
| Heading L | Заголовки страниц |
| Heading M | Заголовки разделов |
| Heading S | Заголовки panels |
| Body L | Длинный читаемый текст |
| Body M | Основной интерфейсный текст |
| Body S | Compact UI |
| Caption | Вспомогательный текст |
| Code | Code или structured output |
| Label | Labels и form labels |

## Правила

- Не масштабировать font-size от viewport width.
- Letter spacing остается 0.
- Hero-size typography не использовать внутри tool panels.
- Code использовать только для structured data, IDs, schemas и logs.
