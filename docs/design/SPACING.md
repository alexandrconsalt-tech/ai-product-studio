# Spacing

## Scale

```text
4, 8, 12, 16, 20, 24, 32, 40, 48, 64, 80, 96
```

## Правила

- Compact controls используют 4-8 px.
- Panels используют 12-16 px.
- Page layout использует 24 px.
- Large empty states используют 32-48 px.
- Не добавлять произвольные spacing values без обновления tokens.
