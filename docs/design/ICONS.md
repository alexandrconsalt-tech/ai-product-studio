# Icon System

## Библиотека

Использовать Lucide.

## Правила

- Icon size по умолчанию: 16 px.
- В toolbar допустим 16-18 px.
- В empty state допустим 24 px.
- Icon-only buttons обязаны иметь `aria-label`.
- Нельзя использовать icon как единственный carrier критического статуса.
- Не создавать custom SVG icon, если есть подходящий Lucide icon.

## Применение

- Navigation: section icons.
- Toolbar: actions.
- Status: optional semantic indicator.
- AI components: AI status и thinking.
